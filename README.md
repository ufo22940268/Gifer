# Gifer
Let your convert video to gif. And you can use this gif to share to WeChat or Weibo.

## Features

1. Select videos from photo.
2. Trim videos.
3. Video can be replay after trimed.
4. Export to GIF.
5. GIF file can be shared to Wechat and webo.
